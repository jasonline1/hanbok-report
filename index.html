혼주한복 준비 가이드

대여 vs 맞춤 vs 맞춤대여

일반 대여

15~30만원
  • 준비기간: 1~2주
  • 핏: 100% 안 맞을 수 있음
  • 활용: 1회 (반납)
  • 관리: 불필요

맞춤 대여

50~60만원
  • 준비기간: 1~2주
  • 핏: 주요 사이즈 조절
  • 활용: 1회 (반납)
  • 장점: 대여+맞춤 결합

완전 맞춤

70~100만원+
  • 준비기간: 30~40일+
  • 핏: 완벽 맞춤
  • 활용: 여러 번 (명절 등)
  • 관리: 직접 보관

체형별 선택 가이드

키 크고 마른 체형

저고리 길이 길게, 풍성한 치마, 차분한 색상, 당의 추천

키 크고 통통한 체형

축소 색상 선택, 꼭 맞는 저고리, 직선 소매

키 작고 마른 체형

화사한 색, 짧은 저고리, 긴 치마, 동일 톤 매치

키 작고 통통한 체형

어두운 저고리+밝은 치마, 삼회장저고리, 넓은 주름, 긴 고름

체크리스트

  • 포함 항목: 속치마, 속바지, 버선, 신발, 가방, 노리개, 머리장식
  • 피팅비: 계약 시 면제되는지, 별도 비용인지
  • 추가 피팅: 1차 후 수정 가능 여부
  • 파손/오염: 배상 정책 확인
  • 반납: 기한, 방법, 택배 가능 여부
  • 웨딩홀 제휴: 예식장 제휴 업체 확인
  • 양가 색상 조율: 겹치지 않도록
  • 편안함 우선: 예뻐도 불편하면 비추

숨겨진 비용 주의
